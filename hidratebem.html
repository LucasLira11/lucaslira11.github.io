Aplicativo HIDRATE BEM.

Demonstração de competências em desenvolvimento Android nativo.

Android Studio
Kotlin
Jetpack Compose
DataStore
Navigation
Coil
Coroutines

O aplicativo "Hidrate Bem" foi desenvolvido como um projeto para apresentar minhas habilidades em desenvolvimento Android. Utilizando a IDE Android Studio e a linguagem Kotlin, implementei um sistema completo de monitoramento de hidratação diária, com uma interface moderna e intuitiva.

A interface do usuário foi totalmente construída com Jetpack Compose, o framework declarativo do Android, que me permitiu criar telas responsivas e com um design personalizado. Para a persistência de dados do usuário, como a meta diária e o consumo, utilizei o Jetpack DataStore, uma biblioteca moderna que garante a segurança e a performance no armazenamento de dados.

A navegação entre as telas é gerenciada pelo Jetpack Navigation, e a funcionalidade de foto de perfil foi implementada com a biblioteca Coil, demonstrando proficiência no uso de ferramentas de terceiros. A lógica de notificação periódica e o reset diário de consumo foram feitos com AlarmManager e BroadcastReceiver, mostrando conhecimento em APIs de sistema para operações em segundo plano.

Este projeto demonstra minha capacidade de criar um aplicativo do zero, desde a arquitetura de código (ViewModel, Repository) até a implementação de uma UI moderna e funcionalidades avançadas. É uma prova de que consigo levar uma ideia do conceito à distribuição.

O aplicativo ainda não está disponível na Google Play Store.

Baixar APK

Para instalar o APK, pode ser necessário habilitar a instalação de aplicativos de "Fontes Desconhecidas" nas configurações do seu dispositivo Android.
